# Code aus VS Code auf GitHub pushen

1) Terminal in VS Code öffnen (`Ctrl + Shift + ö` oder Ansicht → Terminal).
2) Sicherstellen, dass du im Projektordner bist:
   cd D:\Roland\FuturaVote
3) Status prüfen (optional):
   git status
4) Änderungen vormerken:
   git add .
5) Commit erstellen (Kurze Nachricht anpassen):
   git commit -m "Update"
6) Push zu GitHub (Branch: master):
   git push

Hinweise:
- Remote ist gesetzt auf: https://github.com/Gummi-Backe/futuravote
- Falls nach Login gefragt wird, GitHub-Daten eingeben oder Token nutzen.
- Nach dem Push baut Vercel automatisch und veröffentlicht die Seite.
